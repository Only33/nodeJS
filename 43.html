1. nodeJS 是什么？ 是一个基于 Chrome V8 引擎的 JavaScript 运行环境；

    javaScript 包括：三部分

                1. ECMAscript-----规定语法规范{
                例如：var 、 function 、if
            }
            2. DOM-----document
            3. BOM-----window

    node中，没有页面，没有样式（和HTML、css无关）

  1. 使用： 
       1. 直接新建的文件 后缀为js
   2. 接着可以直接写 js 代码
   3. 书写后的代码，是通过控制台----cmd 命令行来执行的
   4. 在当前新建的 js 文件的 文件夹里面打开 控制台（cmd),快捷键是 Windows键+R（字母键），这样就打开控制台
   5. 在 cmd 命令行输入 node 文件名 ，就可以，控制台会执行你的当前目录下的 js 文件里面的代码
   6. 补充快捷方式：命令行输入node 后 文件名可以直接打头的第一个字，然后按 tab 键，会自动补全文件名称

  1. 构造 http 服务：
               1. 引入一个核心模块： http
           2. 创建一个服务器和创建一个服务：var server = http.createServer()
           3. 通过服务器响应一些数据给浏览器：response.write('<h1>this is the server</h1>')
           4. 每一次请求过来以后服务器必须告诉浏览器，当前请求已经完成：response.end()
           5. 绑定ip地址和端口：listen有三个参数：
                                                 第一个参数：端口号
                                                 第二个参数：ip地址（可以省略），默认是本机：127.0.0.1
                                                 第三个参数：服务器开启成功后的回调函数
           6. 创建一个响应服务后，cmd 命令行已经响应 ‘服务已开启’，那么想在执行命令 按 Ctrl + 两次 C键 就可以了

              例子：一个响应服务
                    var http = require('http');
                    var server = http.createServer();
                                    // 回调函数里面的参数：自定义
                                    //      第一个参数：浏览器发送给服务器请求的数据
                                    //      第二个参数：服务器发送给浏览器的请求数据
                    server.on('request',function(request,response){
                        // writeHead 两个参数：第一个为状态码，  第二个参数为类型
                        response.writeHead(200,{
                            'Content-Type':'text/html;charset=utf-8'
                        });
                            //响应数据
                        response.write('<h3>hello</h3>');
                        response.write('<p>请求内容显示</p>');
                        //结束本次请求
                        response.end();
                    });
                    // 监听端口 127.0.0.1:1000
                    server.listen(1000,function(){
                        console.log('服务开启成功');
                    });
